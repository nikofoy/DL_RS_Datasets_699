# 项目简介

<div align="center">
  <h2>遥感开源数据集收集项目</h2>
  <p>创建于2024年11月14日</p>
</div>

本项目旨在收集和分享遥感开源数据集。如果你觉得这些资源对你有帮助，欢迎Star，也欢迎大家补充更多的数据集！
该项目长期更新，若有遗误或不慎涉及侵权，烦请评论或留言联系~
